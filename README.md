# FatigueDetector
Python script able to detect eyes blink.

# Enviroment 
- **Create a enviroment with conda:**
$ conda create --name <env>
- **Clone the repository** 
- ** Install all packages with conda:**
$ conda install -c conda-forge --file requirements.txt
